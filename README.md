    Project name- JobScribe

    A MERN application for tracking jobs and optimizing resumes with help of AI suggestions for individual job roles.
    Responsive for Mobile and Desktop

 
    User can-
    -Register and Log in 
    -Create,Update, Delete job
    -Upload Your Resume(PDF only)
    -Get suggestions from the Gemini AI Model
    -Track Jobs on dashboard which User have applied
    -See the status of interview,total jobs and applied jobs on dashboard screen


    Technologies Used-    
    Frontend- React, Tailwind CSS 
    Backend-Node.js, Express, Gemini API, Redis
    Database- MongoDB
    
    Folder Structure-
    -backend 
    -frontend
    
    
    Environment Variables-
    create a ".env" file in the "/backend" folder and add the following:
    
    MONGOdb_URL=your_mongodb_connection_string
    REDIS_HOST=your_redis_host
    REDIS_PASSWORD=your_redis_password
    REDIS_PORT=your_redis_port
    JWT_SECRET_KEY=your_jwt_secret
    GEMINI_API_KEY=your_gemini_api_key
